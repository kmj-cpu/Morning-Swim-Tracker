# Morning-Swim-Tracker
## 프로젝트 이름 
아침 수영 마일리지 분석기
## 한 문장 설명
불규칙한 기상 시간과 수영 거리르 매일 기록하고, 이전 주와 비교해 증감률과 평균을 계산해주는 프로그램입니다.
## 풀려는 문제 또는 만들려는 이유
아침마다 수영을 하려고 마음먹지만 기상시간이 오락가락하여 매일 달성하는 수영거리가 크게 다릅니다.
어떤 날은 500m,, 어떤 날은 1000m를 수영하는 등 불규칙한 패턴 때문에 내가 얼마나 자주 일찍 일어나는지, 저번 주보다 이번주가 얼마나 발전했는지 직관적으로 파악하기가 어렵습니다. 
따라서 매일의 기상 시간과 운동 거리를 데이터로 누적하고 이를 분석하여 꾸준한 운동과 기상 루틴 정착을 위한 확실한 동기부여를 얻고자 합니다.
## 핵심 기능 3가지
* **일일 기록 입력 및 출력**: 당일의 기상 시간과 수영 거리(m)를 입력받고, '기록완료! 오늘 {time}에 일어나서 {dostance}m를 수영했습니다.'와 같이 확인 메시지를 출력합니다.
* **주간 통계 연산 및 영구 저장**: 입력받은 기록들을 누적하여 평균 기상 시간과 수영 거리 합계를 계산하며, 프로그램을 껐다 켜도 데이터가 유지되도록 파일 형태로 저장하고 불러옵니다.
* **이전 주 대비 증감률 비교**: 파일에 저장된 이전 주 데이터와 이번 주 데이털르 대조하여 수영 거리 증감률(%)을 계산해줍니다.
## 예상 사용자
나 자신 및 매일 아침 수영 기록을 체계적으로 추적하고 발전 과정을 확인하고 싶은 사람들
## 결과물 형태
CLI
## 사용할 Python 개념
* **변수와 제어문**: 기상 시간과 수영 거리를 입력받아 변수에 할당하고, 'for'문을 이용해 이전기록들을 순회하며 합계와 평균을 계산합니다.
* **리스트와 딕셔너리**: 오늘 하루의 단위 기록(날짜, 기상시간, 수영거리)을 딕셔너리로 묶고, 이를 전체 리스트에 차곡차곡 누적 저장하여 데이터를 구조화합니다.
* **함수**: 데이터 입력을 받는 부분, 통계를 연산하는 부분 등을 각각의 함수로 분리하여 코드의 가독성을 높입니다.
* **파일 입출력**: 기록된 리스트 데이터를 '.txt'나 '.csv'파일로 컴퓨터에 내보내고, 프로그램을 다시 켤 때 기존데이터를 불러오는 기능을 추가로 학습하여 구현합니다.
