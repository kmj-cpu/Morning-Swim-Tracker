# Morning-Swim-Tracker
## 프로젝트 이름 
아침 수영 마일리지 분석기
## 한 문장 설명
불규칙한 기상 시간과 수영 거리르 매일 기록하고, 이전 주와 비교해 증감률과 평균을 계산해주는 프로그램입니다.
## 풀려는 문제 또는 만들려는 이유
아침마다 수영을 하려고 마음먹지만 기상시간이 오락가락하여 매일 달성하는 수영거리가 크게 다릅니다.
어떤 날은 500m,, 어떤 날은 1000m를 수영하는 등 불규칙한 패턴 때문에 내가 얼마나 자주 일찍 일어나는지, 저번 주보다 이번주가 얼마나 발전했는지 직관적으로 파악하기가 어렵습니다. 
따라서 매일의 기상 시간과 운동 거리를 데이터로 누적하고 이를 분석하여 꾸준한 운동과 기상 루틴 정착을 위한 확실한 동기부여를 얻고자 합니다.
## 핵심 기능 3가지
일일 기록 입력 및 출력 : 당일의 기상 시간과 수영 거리(m)를 입력받고, '기록완료! 오늘 {time}에 일어나서 {dostance}m를 수영했습니다.'와 같이 확인 메시지를 출력합니다.
