# 🏊‍♂️ Morning-Swim-Tracker: 아침 수영 마일리지 분석기

> **"불규칙한 기상 시간과 수영 거리를 매일 기록하고, 이전 주와 비교해 증감률과 평균을 계산해 주는 프로그램입니다."**

---

## 🤔 기획 배경 및 목적
저는 매일 아침 **7시 수영 루틴**을 실천하고 있으며, **라이프가드 및 수상안전강사 자격증**을 보유할 정도로 수영에 대한 열정이 강합니다. 

하지만 의지만으로는 제어하기 힘든 '기상 시간'의 미세한 차이가 당일 수영 거리(500m~1000m)를 결정짓는 핵심 변수가 되고 있으며, 이를 체계적으로 관리하지 못해 운동 성과를 객관적으로 파악하기 어려운 실정입니다. 

이에 저의 전공 분야인 **통계적 관점**을 접목하여 불규칙한 기상 패턴과 운동량 데이터를 누적 분석하고자 합니다. 단순한 기록을 넘어 **데이터 기반의 피드백**을 통해 수영 루틴을 단단히 정착시키고, 지속적인 성장을 위한 동기부여를 얻는 것이 본 프로젝트의 핵심 목적입니다.
---

## ✨ 핵심 기능 3가지

1. **📝 일일 기록 입력 및 출력**
   * 당일의 기상 시간과 수영 거리(m)를 입력받습니다.
   * `기록 완료! 오늘 {time}에 일어나서 {distance}m를 수영했습니다.` 와 같이 직관적인 확인 메시지를 출력합니다.

2. **💾 주간 통계 연산 및 영구 저장**
   * 입력받은 기록들을 누적하여 평균 기상 시간과 수영 거리 합계를 계산합니다.
   * 프로그램을 껐다 켜도 데이터가 유지되도록 파일 형태로 저장하고 불러옵니다.

3. **📈 이전 주 대비 증감률 비교**
   * 파일에 저장된 이전 주 데이터와 이번 주 데이터를 대조하여 수영 거리 증감률(%)을 계산합니다.

## 🛠️ 사용할 Python 핵심 개념

* **🔀 변수와 제어문**
  기상 시간과 수영 거리를 입력받아 변수에 할당하고, `for`문을 이용해 이전 기록들을 순회하며 합계와 평균을 계산합니다.
* **📦 리스트와 딕셔너리**
  오늘 하루의 단위 기록(날짜, 기상 시간, 수영 거리)을 `딕셔너리(Dictionary)`로 묶고, 이를 전체 `리스트(List)`에 차곡차곡 누적 저장하여 데이터를 구조화합니다.
* **⚙️ 함수**
  데이터 입력을 받는 부분, 통계를 연산하는 부분 등을 각각의 `함수(Function)`로 분리하여 코드의 가독성과 유지보수성을 높입니다.
* **📂 파일 입출력**
  기록된 리스트 데이터를 `.txt`나 `.csv` 파일로 내보내고, 프로그램을 다시 켤 때 기존 데이터를 불러오는 기능을 추가로 학습하여 구현합니다.
